# pigeon-task

**Description: Simple email formatting webapp for task schedules.**

**Demo: https://pigeontask.com/**

**Tech: JS, HTML, CSS**

# How to use
1. Fill out task details
2. Preview to check formatting
3. Enter email in form, and send off
